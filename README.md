## 🚀 Featured Projects

### 🏃 Human Activity Recognition System
**2024 – Present** | `Python` `MediaPipe` `OpenCV`

A real-time human activity recognition system that detects body landmarks from live webcam input using MediaPipe Pose and OpenCV.

- Classifies six activities — Standing, Walking, Running, Sitting, Boxing, and Clapping — using joint-angle calculations and temporal pose tracking across consecutive frames
- Implements email notification alerts for anomalous activities (Boxing, Running)
- Improved prediction stability with multi-frame smoothing using deque-based history buffers

**Key learning outcomes:** Real-time computer vision pipelines, pose estimation, temporal signal smoothing, event-driven alerting systems

🔗 Repository: [https://github.com/kmaheshbabu75/Human-Activity-Recognition-using-MediaPipe]

### 📄 AI-Based Web Content Summarizer with Q&A and Audio
**2025** | `Python` `Streamlit` `HuggingFace`

A web application that fetches and extracts article content from URLs and generates abstractive summaries using the DistilBART NLP model.

- Implemented a question-answering feature using DistilBERT to retrieve relevant answers from summarized content, with persistent session-based Q&A history
- Added text-to-speech audio generation using gTTS for audio playback and download
- Developed an interactive multi-section user interface using Streamlit

**Key learning outcomes:** NLP model integration, web scraping, transformer-based summarization/QA pipelines, building interactive data apps

 🔗 Repository: [https://github.com/kmaheshbabu75/An-AI-Based-Web-Content-Summarization-System-with-Question-Answering-and-Audio-Generation]
