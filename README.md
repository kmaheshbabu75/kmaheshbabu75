## Featured Projects

### 🏃 Human Activity Recognition System
- Tech: Python, MediaPipe, OpenCV
- Real-time activity recognition from live webcam input using MediaPipe Pose and joint-angle calculations, classifying six activities (Standing, Walking, Running, Sitting, Boxing, Clapping)
- Email alert notifications for anomalous activities (Boxing, Running) with multi-frame smoothing via deque-based history buffers for stable predictions

### 📄 AI-Based Web Content Summarizer with Q&A and Audio
- Tech: Python, Streamlit, HuggingFace, BeautifulSoup, gTTS
- Web app that extracts article content from URLs and generates abstractive summaries using the DistilBART NLP model
- Question-answering feature powered by DistilBERT with persistent session-based Q&A history, plus text-to-speech audio generation via gTTS
